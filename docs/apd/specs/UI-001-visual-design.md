# UI-001: ビジュアルデザインの洗練

## ユーザーストーリー

**誰が**: AWS SA / Sales
**何を**: 商談やプレゼンの場でお客様に見せても恥ずかしくない、プロフェッショナルなデザインのサイトを使いたい
**なぜ**: ツールの見た目がお客様への信頼感に直結し、提案の説得力に影響するため

## 受け入れ条件

### AC-1: AWSブランドとの親和性

```
Given: ユーザーがサイトにアクセスする
When: ページが表示される
Then: AWSのカラーパレット（Squid Ink、Orange等）をベースとしたデザインである
And: プロフェッショナルで信頼感のある見た目である
```

### AC-2: テーブルの視認性

```
Given: テーブルが表示されている
When: ユーザーがデータを閲覧する
Then: 世代ごとの色分けが適切で、視覚的にグループが識別できる
And: 重要なデータ（価格、VRAM、性能）が目立つ
And: 行ホバー時のハイライトが滑らかである
```

### AC-3: タイポグラフィ

```
Given: ページが表示されている
When: ユーザーがコンテンツを閲覧する
Then: フォントが読みやすく、数値データは等幅で整列している
And: ヘッダー・本文・注記のサイズ階層が明確である
```

### AC-4: レスポンシブデザイン

```
Given: 様々なデバイスでアクセスする
When: PC、タブレット、モバイルで表示する
Then: 各デバイスで適切にレイアウトされる
And: テーブルは横スクロール可能で、データが途切れない
```

### AC-5: ライト/ダークモード切替

```
Given: ユーザーがサイトにアクセスする
When: ページが読み込まれる
Then: OSのテーマ設定（prefers-color-scheme）に応じてライト/ダークモードが自動選択される
And: 手動で切り替えるトグルUIが提供される
And: ダークモードはSquid Inkベース、ライトモードは明るい背景で視認性を確保
```

## UI記述

- ダークモード: 現状のAWSカラーパレットベース（Squid Ink背景）
- ライトモード: 白系背景にAWSカラーのアクセント、プロジェクター投影にも適した配色
- 世代別の色分け（Blackwell=紫、Hopper=青、Ada=ティール、Ampere=オレンジ、Turing=緑、Volta=グレー）
- 数値は右揃え・等幅フォント
- 適切な余白とボーダーで情報の境界を明確に
- テーマ切替トグルはヘッダーに配置（言語切替の近く）

## コンテキスト境界

- **Inputs**: CSS変数（カラーパレット）、デバイス情報（viewport）、OSテーマ設定（prefers-color-scheme）、ユーザーのテーマ選択
- **Outputs**: スタイル適用されたUI
- **Dependencies**: REF-001（テーブル構造）、CAL-001（計算ツールUI）、I18N-001（文言表示）
