# CAL-001: 簡易コスト計算ツール

## ユーザーストーリー

**誰が**: AWS SA / Sales / GPUを検討中のお客様
**何を**: インスタンスタイプと台数を入力して、月額の概算コストを算出したい
**なぜ**: 商談中やインスタンス選定時に、ざっくりとしたコスト感を素早く把握するため

## 受け入れ条件

### AC-1: 計算ツールの表示

```
Given: ユーザーがサイトにアクセスする
When: ページが読み込まれる
Then: コスト計算ツールがテーブルとは別のセクションとして表示される
```

### AC-2: 入力項目

```
Given: コスト計算ツールが表示されている
When: ユーザーが計算を行う
Then: 以下の入力項目が利用できる
  - インスタンスタイプの選択（GPU_DATAから選択肢を生成）
  - 台数の入力（数値、デフォルト: 1）
```

### AC-3: 月額コスト算出

```
Given: ユーザーがインスタンスタイプと台数を入力している
When: 入力値が変更される
Then: 月額コスト（時間単価 × 720時間 × 台数）がリアルタイムに表示される
And: On-Demand月額とCapacity Blocks月額が並べて表示される
And: 通貨はUSD（$）で表示される
```

### AC-4: CB専用インスタンス

```
Given: CB専用インスタンス（On-Demand価格がないもの）を選択する
When: 月額コストが表示される
Then: On-Demand月額は「CB専用」と表示される
And: Capacity Blocks月額のみ計算される
```

### AC-5: 計算精度の注記

```
Given: コスト計算ツールが表示されている
When: ユーザーがツールを閲覧する
Then: 「概算値です。RI/Savings Plans等の割引は含まれません。正確な料金はAWS公式をご確認ください」という注記が表示される
```

## UI記述

- テーブルの下部に配置
- インラインで即座に結果表示（ページ遷移なし）
- 入力変更時にリアルタイム計算（ボタン押下不要）
- On-DemandとCBの月額を並べて表示
- コンパクトなフォームレイアウト

## コンテキスト境界

- **Inputs**: GPU_DATA配列（価格情報）、ユーザー入力（インスタンス選択、台数）
- **Outputs**: 月額概算コスト（On-Demand / CB、USD）
- **Dependencies**: REF-001（GPU_DATAを共有）
