# I18N-001: 多言語対応（日本語・英語）

## ユーザーストーリー

**誰が**: 英語話者のお客様、グローバルチームのメンバー
**何を**: サイトのUI文言を英語で閲覧したい
**なぜ**: 海外のお客様やグローバルチームと共有する際に、言語の壁なく情報を参照するため

## 受け入れ条件

### AC-1: 言語切替UI

```
Given: ユーザーがサイトにアクセスする
When: ページが読み込まれる
Then: 言語切替のUI要素（日本語/English）が表示される
```

### AC-2: デフォルト言語の自動判定

```
Given: ユーザーが初めてサイトにアクセスする（localStorageに言語設定なし）
When: ページが読み込まれる
Then: navigator.languageでブラウザの言語設定を取得する
And: 日本語（ja）の場合は日本語、それ以外は英語で表示する
```

### AC-3: 言語切替の動作

```
Given: 日本語で表示されている
When: ユーザーが「English」を選択する
Then: UI文言（ヘッダー、カラム名、ラベル、注記等）が英語に切り替わる
And: ページリロードなしで即座に切り替わる
And: データ値（数値、インスタンス名等）は変更されない
```

### AC-4: 翻訳対象

```
Given: 言語が切り替えられる
When: 英語表示に切り替える
Then: 以下が翻訳される
  - ページタイトル・ヘッダー
  - テーブルのカラムヘッダー
  - 世代ラベル
  - 注記・フッター
  - コスト計算ツールのラベル・メッセージ
And: 以下は翻訳されない
  - インスタンスタイプ名（p5.48xlarge等）
  - GPUモデル名（H100等）
  - 数値データ
  - 価格データ
```

### AC-5: 言語設定の永続化

```
Given: ユーザーが言語を手動で切り替えた
When: 同じブラウザで再度サイトにアクセスする
Then: localStorageに保存された言語設定が優先される
And: ブラウザの言語設定より優先して適用される
```

## UI記述

- 言語切替はヘッダー右上に配置
- トグルスイッチまたはドロップダウンで切り替え
- 現在の言語が視覚的に明示される

## コンテキスト境界

- **Inputs**: navigator.language（ブラウザ言語）、localStorage（言語設定の永続化）、ユーザーの言語選択
- **Outputs**: 翻訳されたUI文言
- **Dependencies**: REF-001（カラムヘッダー）、CAL-001（ラベル・メッセージ）
